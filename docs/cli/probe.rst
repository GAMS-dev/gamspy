gamspy probe
============

Probes the node to get information about the node.

Usage
-----

::

  gamspy probe -o info.json  

.. list-table::
   :widths: 20 20 20 40
   :header-rows: 1

   * - Option
     - Short
     - Default
     - Description
   * - -\-output 
     - -o
     - 
     - Output path to dump probed information.
