gamspy run
==========

Runs the GAMS MIRO application.

Usage
-----

::

  gamspy run miro [OPTIONS]  


.. list-table::
   :widths: 20 20 20 40
   :header-rows: 1

   * - Option
     - Short
     - Default
     - Description
   * - -\-path 
     - -p
     - 
     - Path to your GAMS MIRO installation.
   * - -\-model 
     - -g
     - 
     - Path to your model.
