gamspy uninstall
================

Uninstalls an existing solver or a license from the GAMSPy installation.

Usage
-----

::

  gamspy uninstall <solver|license> [OPTIONS]  

.. list-table::
   :widths: 20 20 20 40
   :header-rows: 1

   * - Option
     - Short
     - Default
     - Description
   * - -\-skip-pip-uninstall 
     - -a
     - 
     - Skips the pip uninstall command in case the package was manually deleted.
